This backend for **Curry Craft** is built with **Node.js** and **Express**, utilizing **MongoDB** and **Mongoose** for robust data management. It features a secure **JWT-based authentication** system with access and refresh token logic to protect user data. I implemented core functionalities including a **Cloudinary-integrated** video upload system, recipe management, and an advanced search logic for ingredient-based discovery. The project also includes social engagement features like comments, likes, and a subscription model, all powered by optimized **aggregation pipelines** for high performance.
